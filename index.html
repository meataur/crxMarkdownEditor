Features and Tips

  • Support drag-and-drop to present files attached to markdown document.
  • Support hashing for preventing duplication of resource filenames.
  • Press save button to save editor's content as a file.
  • Enable easy blog-posting through browser setting to ask where to save file before downloading.
  • Jekyll standalone installer is available at github.com/ChangUk/jekyll-standalone.

Keyboard Shortcuts

  • Find text matches: Ctrl + F
    • Find next: Enter
    • Find previous: Shift + Enter
  • Replace all: Ctrl + H
  • Jump to line: Alt + G
