Author

ChangUk, Park (changuk.github.io)

Features

  • Support drag-and-drop to display attachments on the editor.
  • Provide hash functions for preventing attachment filename duplicates.
  • Support import/export to cloud storage such as Google Drive or Github.
  • Provide Jekyll Launcher program to easily execute Jekyll engine on localhost.

Keyboard Shortcuts

  • Open markdown document from local system: Ctrl + O
  • Save markdown document as local file: Ctrl + S
  • Find text matches: Ctrl + F
    • Find next: Enter
    • Find previous: Shift + Enter
  • Replace all: Ctrl + H
  • Jump to line: Alt + G
  • Return to editor on expanded-viewer mode: Esc
